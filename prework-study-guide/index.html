HTML

  • The head element contains information about the webpage.
  • The body element resresents the visible content showne to the user.
  • The UL element means unordered list
  • The LI element means list item

CSS

  • CSS means cascading style sheets
  • CSS is used to give color and style to web pages
  • CSS is used with Reddit extensively
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class selector differs from * in that it specifies what specific items will have the CSS style attached to them.
